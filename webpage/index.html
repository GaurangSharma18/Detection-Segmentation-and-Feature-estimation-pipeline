Abstract
Deep Learning has led to significant advances in computer vision, making perception an important component in many fields such as robotics, medicine, agriculture, remote sensing, etc. As Deep Learning requires large amounts of data, a well-defined pipeline for labeling and augmentation is needed as well. Several data labeling and augmentation strategies exist, supporting numerous computer vision tasks with dedicated formats and annotation types, such as bounding boxes, polygons, and keypoints. Image annotation types can even be combined into a single data format to benefit approaches such as multi-task models. However, generating a combination of such annotations is not straight-forward, as no single solution or tool provides these functionalities. To our knowledge, no available labeling tool supports the export functionality for a combined format, and no augmentation library supports transformations for the combination of polygon segmentation, bounding boxes, and keypoints. In this paper, we present a novel approach that spans from labeling to augmentation. The paper also provides additional functionality to the tools/libraries used, which enables them to perform valuable tasks that require object detection, segmentation, and feature extraction in a single application. Finally, The functionality and the use of this pipeline in various computer vision and robotics tasks are presented in the paper.
